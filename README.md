# AIO

This project arose as a playground to learn the basics of the Object Oriented Programing for the Python language. It includes several basic programing challenges such us: Tic Tac Toe, Saddle Point, Dungeon Crawler, Conway's Game of Life, etc.

## ARCHITECTURE

1. [Core](https://github.com/javierpardollama/AIO/tree/main/core)

In this project, several core utilities are provided.

1. [Shared](https://github.com/javierpardollama/AIO/tree/main/shared)

In this project, several shared utilities are provided.

2. [Conway](https://github.com/javierpardollama/AIO/tree/main/conway)

In this project, Conway's Game Of Life itself is provided.

3. [Dungeon](https://github.com/javierpardollama/AIO/tree/main/dungeon)

In this project, Dungeon Crawler itself is provided.

4. [Saddle](https://github.com/javierpardollama/AIO/tree/main/saddle)

In this project, Saddle Point itself is provided.

4. [Tic Tac](https://github.com/javierpardollama/AIO/tree/main/tiktak)

In this project, Tik Tak Toe itself is provided.

## GETTING STARTED

To compile and run this project, it is recommended to use the following tools, which were used to build and test this project:

1. [Python](https://www.python.org/)

This framework is used to build, compile and serve everything related to this project

1. [PyCharm](https://www.jetbrains.com/pycharm/)

This IDE is used to code and debug everything related to this project
